# DEFRAG Product Overview

DEFRAG is a personal and relational clarity product.

Core promise:
- Help people understand themselves
- Help people read what may be active right now
- Help people make better sense of relationship dynamics

Core surfaces:
- Baseline
- Now
- Relationships
- Learn

Product rules:
- Calm, premium, plain-language tone
- No therapy voice
- No mystical-first framing
- No generic productivity framing
- One coherent system across public pages, onboarding, and app runtime

Current implementation direction:
- Single Next.js app
- Shared layout primitives
- Shared copy registry
- Ticket-by-ticket implementation only
